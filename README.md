# Azure PowerShell Automation Project

## Objective
Automate Azure resource management using PowerShell on Linux and maintain version control using Git and GitHub.

## Tools Used
- Azure Cloud
- Linux VM (Ubuntu)
- PowerShell Core
- Git & GitHub
- GitHub Actions (CI/CD)

## Project Tasks Completed
- Installed PowerShell on Linux VM
- Created Azure automation scripts:
  - Resource Group creation and listing
  - Storage account creation and listing
- Tested scripts in Azure environment
- Implemented version control using Git
- Pushed code to GitHub repository
- Created CI pipeline using GitHub Actions

## CI/CD Pipeline
- Automatically runs PowerShell scripts on every push
- Validates script execution
- Ensures code quality and automation testing

## Result
End-to-end automation workflow for Azure using PowerShell + Git + CI/CD pipeline successfully implemented.
